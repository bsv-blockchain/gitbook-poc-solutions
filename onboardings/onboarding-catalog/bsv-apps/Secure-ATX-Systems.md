# Secure ATX Systems Onboarding Guide

Secure ATX is a browser-based key management solution using Shamir's Secret Sharing Scheme and QR code paper wallets.

## Key Features
- Split secret into multiple shares, set threshold for recovery
- Generate printable QR codes
- All cryptographic operations are browser-local

## Getting Started

1. Visit [https://secure.atx.systems/](https://secure.atx.systems/)
2. Click **Backup** to split key
   - Set number of shares, threshold
   - Generate and print/share QR codes
3. Click **Recover** to reconstruct secret
   - Scan minimum shares and the key is restored

## Tips
- Distribute shares to trusted parties/locations
- Validate shares before critical recovery

## Troubleshooting
- Recovery fails: Ensure all shares are correct and minimum threshold is met
- Print QR codes for true offline backup
